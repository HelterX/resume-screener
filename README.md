# Resume Screener 📄

An AI-powered resume screening tool that compares your resume against a job description and gives you an instant match score, keyword analysis, and improvement tips.

## Features

- Paste any job description + resume text
- Instant match score (0–100%) with animated ring
- Matched vs missing keyword breakdown
- Tailored improvement tips
- Fully responsive (mobile + desktop)
- No backend, no API key needed

## Built With

- HTML, CSS, JavaScript (vanilla — no frameworks)
- Smart keyword extraction & weighted scoring algorithm
- Hosted on GitHub Pages
